# Movie Tickets

##### Epicodus Section: JavaScript OO - Movie Tickets

#### By Grady L Shelton and Dominic Brown

## Description

This application is a practice of constructors and prototypes using JavaScript to create a simple Address Book app.

## Specifications

* Program will take user input to select name of a movie they wish to watch, ask for age, and what time of day they would like to watch.
  * **Example Input:** "Mad Max, Age: 18, Time of day: 9:00pm"
  * **Example Output:** "Mad Max, Age: 18, Time of day: 9:00pm"

* Program will decide a custom ticket price based on user input. If user is too young an alert will appear indicating as such.
  * **Example Input:** "Mad Max, Age: 18, Time of day: 9:00pm"
  * **Example Output:** "$14.00 Have fun!"

## Setup

Install address-book by cloning this repository.

## Technologies Used

JavaScript and jQuery
